The
ASCII
character
set
is
the
most
common
format
for
English
language
text
files
and
is
generally
assumed
to
be
the
default
file
format
in
many
situations
For
accented
and
other
non-ASCII
characters
it
is
necessary
to
choose
a
character
encoding.
In
many
systems
this
is
chosen
on
the
basis
of
the
default
locale
setting
on
the
computer
it
is
read
on
Common
character
encodings
include
ISO
for
many
European
languages
